# <a name="azure-ad-domain-services---contact-us"></a>Služba Azure AD Domain Services - nás kontaktovat
## <a name="contact-the-product-team"></a>Obraťte se na tým produktu
Pokud máte problémy s vaší spravované domény, zkontrolujte, pokud podle kroků uvedených v [Průvodce odstraňováním potíží](active-directory-ds-troubleshooting.md) vyřešte problém. Pokud máte pořád problémy, klidně nás kontaktovat.

Může e-mailem nás na adrese: [zpětnou vazbu Azure AD Domain Services](mailto:aaddsfb@microsoft.com).

Ujistěte se, zahrnuje následující, takže jsme problém prozkoumat.

* **ID nebo adresář ID klienta** pro váš adresář Azure AD. ID klienta je identifikátor GUID, uvidíte v poli ID adresáře na [stránku vlastností pro váš adresář](https://ms.portal.azure.com/#blade/Microsoft_AAD_IAM/ActiveDirectoryMenuBlade/Properties)
* **Název domény DNS** jste nakonfigurovali pro vaší spravované domény služby AAD Domain Services.

## <a name="provide-feedback"></a>Poslat názor
Uvítáme vaše názory o Azure AD Domain Services pomocí  **[kanál Azure Active Directory User Voice](https://feedback.azure.com/forums/169401-azure-active-directory/)**.

Ujistěte se, že ve můžete pre čekání otázku nebo zpětnou vazbu s slova **'AADDS'**, pro dosáhne nás.
