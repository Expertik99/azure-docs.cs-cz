# <a name="azure-active-directory-b2b-collaboration-invitation-redemption"></a>Azure Active Directory s B2B spolupráce pozvánku k uplatnění kódu.

## <a name="azure-ad-and-microsoft-account-users"></a>Azure AD a uživatelé s účtem Microsoft
Pro uživatele s existující účet Azure AD nebo účty Microsoft je stejně snadná jako přihlášení prostředí uplatnění kódu.

## <a name="social-id-user-first-time-redemption"></a>Sociální ID uživatele prvního uplatnění kódu.
Spolupráce B2B Azure AD umožňuje snadno pro všechny e-mailovou adresu, který se má použít pro uplatnění. Podívejte se na uplatnění zkušeností při jiných společností než Microsoft e-mailová adresa se používá pro spolupráci B2B. Tento tok uplatnění totiž složitější, možná budete muset vytvořit účet v době uplatnění. Najdete ho v následujícím videu:

> [!VIDEO https://channel9.msdn.com/Blogs/Azure/b2b-collaboration-redemption/Player]
