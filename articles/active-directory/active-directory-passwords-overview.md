# <a name="azure-ad-self-service-password-reset-for-the-it-professional"></a>Azure AD samoobslužného resetování hesla pro IT profesionály

Azure Active Directory (Azure AD) samoobslužné resetování hesla (SSPR) umožňuje uživatelům resetovat vlastní hesla na své vlastní, kdy a kde potřebují, při umožňuje správcům kontrolu nad jak získá resetování hesla. Uživatelé se již muset volat na HelpDesk jenom k resetování hesla.

* **Změna hesla pomocí samoobslužné služby** – uživatel zná heslo, ale chce změnit na novou.
* **Resetování hesla pomocí samoobslužné služby** – uživatel nemůže se přihlásit a chce resetovat heslo pomocí jednoho nebo více z následujících metod ověřování ověřen.
   * Textová zpráva na ověřené mobilní telefon
   * Telefonní hovor ověřené telefon mobilní telefon nebo office
   * E-mailem ověřené sekundární e-mailový účet
   * Odpovědi na své bezpečnostní otázky
* **Odemknutí účtu samoobslužné služby** – uživatele se nepodařilo přihlásit se pomocí hesla byl uzamčen a chcete odemknout svůj účet bez zásahu správce pomocí své metody ověřování.

## <a name="why-choose-azure-ad-self-service-password-reset"></a>Proč zvolit Azure AD samoobslužného resetování hesel

* **Snížení nákladů** jako helpdesk obvykle obnoví odbornou heslo účtu pro 20 % volání podpory IT organizace. 
* **Vylepšení prostředí pro koncové uživatele** a **snížit helpdesk svazku** tím, že koncoví uživatelé power vyřešit problémy s jejich vlastní hesla najednou bez volání technickou podporu nebo otevření žádosti o podporu.
* **Jednotka mobility** jako uživatelům můžete resetovat vlastní hesla ze kdekoli jsou.
* **Zachovat kontrolu** zásad zabezpečení. Správci mohou nadále vynucovat zásady, které ještě dnes.

Pokud je všechno připravené, abyste mohli začít s Azure AD SSPR pomocí našich [rychlý start pokyny](active-directory-passwords-getting-started.md) a rychle uživatelům resetovat vlastní hesla.

## <a name="azure-ad-self-service-password-reset-availability"></a>Dostupnost resetování hesel samoobslužné služby Azure AD

Heslo samoobslužné služby Azure AD vynulování je k dispozici ve třech úrovních v závislosti na vaše předplatné.

* **Azure AD volné** – jenom pro Cloud správci můžou resetovat vlastní hesla.
* **Azure AD Basic** nebo jakoukoli **placené předplatné Office 365** – jenom pro Cloud uživatelům můžete resetovat jejich hesla.
* **Azure AD Premium** – všechny uživatele nebo správce, včetně jenom pro cloud, federovaný, nebo heslo synchronizované uživatele, můžete resetovat vlastní hesla. Místních hesel vyžadují zpětný zápis hesla tak, aby byl povolen.
