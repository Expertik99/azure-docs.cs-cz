# <a name="how-do-information-workers-add-b2b-collaboration-users-to-azure-active-directory"></a>Jak informační pracovníci přidat uživatele spolupráce B2B do Azure Active Directory

Informační pracovníci můžete použít [Panel přístupu aplikace](http://myapps.microsoft.com) k přidání uživatele spolupráce B2B do skupiny a aplikace, které budou spravovat.

Po přidání uživatele guest k adresáři pracovník může buď odesílat uživatele guest přímý odkaz na aplikaci ve sdílených nebo uživatele guest můžete kliknout na adresu URL se v e-mailová pozvánka. Další informace o procesu uplatnění najdete v tématu [uplatnění pozvánku spolupráce B2B](redemption-experience.md).

## <a name="information-workers-adding-b2b-collaboration-users-to-a-group"></a>Informační pracovníci přidávání uživatelů spolupráce B2B do skupiny

Informační pracovníci podobně můžete přidat uživatele spolupráce B2B do přiřazené skupiny, který je povolený pro samoobslužnou správu skupin.
> [!NOTE]
> Uživatelé spolupráce B2B nelze přidat do dynamická skupina nebo skupiny, který je synchronizovaný s místní služby Active Directory.
