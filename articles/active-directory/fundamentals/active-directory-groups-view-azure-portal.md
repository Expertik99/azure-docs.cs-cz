# <a name="view-or-search-for-your-user-groups-in-azure-active-directory"></a>Zobrazení nebo vyhledání skupin uživatelů v Azure Active Directory
Tento článek vysvětluje, jak zobrazit skupiny v Azure Active Directory (Azure AD). Mezi možnosti správy uživatelů Azure AD patří použití skupin k provádění úloh správy, například k přiřazení licencí nebo oprávnění více uživatelům najednou.

## <a name="how-do-i-see-all-the-groups"></a>Jak zobrazím všechny skupiny?
1. Přihlaste se k portálu [Azure Portal](https://portal.azure.com) prostřednictvím účtu, který má k adresáři oprávnění globálního správce.
2. Vyberte **Všechny služby**, do textového pole zadejte **Uživatelé a skupiny** a potom stiskněte **Enter**.

   ![Otevření uživatelů a skupin](./media/active-directory-groups-view-azure-portal/search-user-management.png)
3. V okně **Uživatelé a skupiny** vyberte **Všechny skupiny**.

   ![Otevření okna skupin](./media/active-directory-groups-view-azure-portal/view-groups-blade.png)
4. V okně **Uživatelé a skupiny – Všechny skupiny** můžete přidat nebo odebrat sloupce zobrazení, vyhledat skupinu filtrováním seznamu nebo provádět změny skupin, ke kterým máte dostatečná oprávnění.
