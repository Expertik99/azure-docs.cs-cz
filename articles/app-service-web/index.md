    <h1>Dokumentace k Web Apps</h1>
    <p style="padding-bottom: 0px; max-width: 1050px;">Azure Web Apps umožňuje vytvářet a hostovat weby v programovacím jazyce podle vašeho výběru, aniž by bylo potřeba zabývat se správou infrastruktury. Nabízí automatické škálování a vysokou dostupnost, podporuje systémy Windows a Linux (Preview) a umožňuje automatizované nasazení z Githubu, Team Services nebo libovolného úložiště Git. Naučte se používat Azure Web Apps prostřednictvím našich šablon Rychlý start, výukových kurzů a ukázek.</p>
