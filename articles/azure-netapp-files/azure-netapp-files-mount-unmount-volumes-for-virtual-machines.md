# <a name="mount-or-unmount-a-volume-for-windows-or-linux-virtual-machines"></a>Připojování nebo odpojování svazku pro virtuální počítače s Windows nebo Linux 

Můžete připojit nebo odpojit podle potřeby svazek pro virtuální počítače s Windows nebo Linux.  Pokyny připojení pro virtuální počítače s Linuxem jsou k dispozici souborů NetApp Azure.  

1. Klikněte na tlačítko **svazky** okna a pak vyberte svazek, u kterého chcete připojení systému souborů NFS exportuje. 
2. Klikněte na tlačítko **pokyny pro připojení** z vybraných svazků a pak postupujte podle pokynů.
