# <a name="supported-browsers-and-devices-for-the-azure-portal"></a>Podporovaných prohlížečích a zařízeních pro portál Azure
Můžete spustit [portál Azure](https://portal.azure.com) na všechny plochy moderní, tablet zařízení a prohlížeče.

### <a name="supported-devices"></a>Podporovaná zařízení
Portál Azure spouští skvělé na moderní počítače, počítače Mac a tablety. Pokud potřebujete ke správě prostředků Azure z mobilního zařízení, zkuste Azure mobilní aplikace, která je dostupná na zařízení iPhone a Android. Další informace najdete v tématu [mobilní aplikace Azure](https://azure.microsoft.com/features/azure-portal/mobile-app/).

### <a name="supported-browsers"></a>Podporované prohlížeče
Doporučujeme vám, že používáte nejnovější prohlížeče, který je kompatibilní s operačním systémem. Podporované jsou následující prohlížeče:

* Microsoft Edge (nejnovější verze)
* Internet Explorer 11
* Safari (nejnovější verzi, pouze Mac)
* Chrome (nejnovější verze)
* Firefox (nejnovější verze)
