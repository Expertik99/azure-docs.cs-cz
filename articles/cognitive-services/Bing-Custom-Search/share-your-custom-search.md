# <a name="share-your-custom-search-instance"></a>Sdílet instanci vlastní vyhledávání
Vaše vlastní hledání instance můžete sdílet s kýmkoli pomocí právě e-mailové adresy. Sdílení umožňuje ostatním uživatelům upravit vaše vlastní hledání instance. Přihlaste se k [vlastní vyhledávání](https://customsearch.ai), vyberte instanci vlastní vyhledávání a klikněte na ikonu nastavení (se zobrazí jako stupeň). V **sdílet vaše Instance** zadejte e-mailovou adresu osoby, která má sdílet s vaší instancí a klikněte na tlačítko **sdílet**. E-mailová adresa se zobrazí v seznamu s názvem **instanci sdílet s**. Opakujte pro každou osobu, kterou chcete sdílet pomocí vlastní hledání. 

Pokud chcete přidat jiného uživatele e-mailu do seznamu, není nezbytné pro hledání vlastní účet. Je třeba zaregistrovat vlastní vyhledávání před jejich udělat změny konfigurace. Po sdílení instance s někým, se zobrazí v jejich seznam instancí vlastní vyhledávání. Jenom jedna osoba, můžete upravit instance současně. Pokud se pokusíte upravit instanci, která upravuje někdo jiný, zobrazí se upozornění. Instance je možné sdílet s maximálně 10 uživatelů.

## <a name="stop-sharing"></a>Zastavit sdílení
Chcete-li ukončit sdílení instance s někým, použijte k odebrání e-mailové adresy ze seznamu ikonu odebrat.
