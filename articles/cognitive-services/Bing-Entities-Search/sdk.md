# <a name="bing-search-sdk"></a>Hledání Bing SDK
Ukázky rozhraní API pro Bing Entity hledání zahrnout scénáře který:
1.  Vyhledejte entity, jako je například tní výletních a bohaté informace.
2.  Zpracování rozlišení více tras podmínek pro dotazy s pravděpodobně více tříd Intent.
3.  Vyhledejte místní entity, jako je například restaurace a bohaté informace kolem něj.
4.  Vyhledejte místní podniky, například restaurace a bohaté informace.
5.  Aktivovat chybný požadavek a zpracování chyb.

> Některé sady SDK jsou teď ve GA a jsou čekající změny v dokumentaci.
