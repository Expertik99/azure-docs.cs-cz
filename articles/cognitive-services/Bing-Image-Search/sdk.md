# <a name="bing-search-sdk"></a>Hledání Bing SDK
Ukázky rozhraní API Search bitové kopie Bingu zahrnují senarios který:
1.  Dotaz pro jednoduché bitovou kopii.
2.  Vyhledávání omezení na konkrétní image typy a poměrům stran zobrazující image možností filtrování.
3.  Hledání trendů v různých sociálních sítích v tuto chvíli bitové kopie.
4.  Dotaz na bohaté podrobnosti bitové kopie, jako je například zdroj bitové kopie, text reprezentativní dotazu atd.

Sady SDK vyhledávání Bing zpřístupněte funkce vyhledávání web snadno v těchto programovacích jazycích:

    * Viz také [knihovny .NET](https://github.com/Azure/azure-sdk-for-net/tree/psSdkJson6/src/SDKs/CognitiveServices/dataPlane/Search/BingImageSearch) pro definice a závislosti.
* Začínáme s [ukázky Node.js](https://github.com/Azure-Samples/cognitive-services-node-sdk-samples) 
    * Viz také [Node.js knihovny](https://github.com/Azure/azure-sdk-for-node/tree/master/lib/services/imageSearch) pro definice a závislosti.
* Začínáme s [ukázky Java](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples) 
    * Viz také [Java knihovny](https://github.com/Azure/azure-sdk-for-java/tree/master/azure-cognitiveservices/search/bingimagesearch) pro definice a závislosti.
