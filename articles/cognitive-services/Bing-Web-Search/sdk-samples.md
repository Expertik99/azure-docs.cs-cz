# <a name="bing-web-search-sdk-samples"></a>Sada SDK vyhledávání Bingu webové ukázky

Sady SDK webové vyhledávání Bingu je dostupné v Pythonu, Node.js, C# a Java. Ukázky kódu, požadavky a pokyny pro sestavení jsou k dispozici na Githubu. Jsou popsané následujících scénářích:

* Dotazování jedno slovo a vytiskne název a adresu URL první výsledek pro webové stránky, obrázky, vybrané články a videa.
* Dotazování slovní spojení, zkontrolujte počet výsledků a vytiskne název a adresu URL první výsledek.
* Hledaný výraz dotazu s filtry odpovědi nastavené `news` a tisknout podrobnosti výsledky zpráv.
* Dotazování hledaný výraz s `answerCount` a `promote` parametry, pak vytiskněte podrobnosti výsledků.
