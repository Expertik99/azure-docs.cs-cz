# <a name="sign-up-for-content-moderator"></a>Zaregistrovat se k obsahu moderátora #
Přejděte na [obsahu moderátora zaregistrovat](http://contentmoderator.cognitive.microsoft.com/Account/Signup) stránky. Můžete zaregistrovat pomocí účtu Microsoft, nebo vytvořit účet na webu moderátora obsahu.

![Registrace](images/0-signup-1.PNG)

## <a name="name-your-team"></a>Název vašeho týmu ##
Zadejte název pro váš tým revize. Volitelně můžete pozvat kolegy k připojení vašeho týmu.

![Vytvoření týmu](images/1-Team-Name.PNG)
