# <a name="view-dashboard"></a>Zobrazit řídicí panel #

Klikněte **řídicí panel** KARTĚ pro zobrazení klíčových metrik pro všechny kontrolu obsahu provést v rámci nástroje. Můžete zobrazit celkový, dokončené a v průběhu recenze pro obrázky a text. Se zobrazí metriky pro váš tým výchozí a pro váš subteams.
