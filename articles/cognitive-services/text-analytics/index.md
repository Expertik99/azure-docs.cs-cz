    <h1>Dokumentace k rozhraní Text Analytics API</h1>
    <div class="intro" style="min-width: 200px">
        <p>Text Analytics API je cloudová služba, která poskytuje rozšířené zpracování přirozeného jazyka pro nezpracovaný text a zahrnuje tři hlavní funkce: analýzu mínění, extrakci klíčových frází a rozpoznávání jazyka. Pomocí našich šablon Rychlý start, kurzů a ukázek se naučíte různými způsoby analyzovat obsah.</p>
    </div>
