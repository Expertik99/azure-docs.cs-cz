# <a name="how-to-sign-up-for-the-microsoft-translator-speech-api"></a>Postup registrace pro rozhraní API pro rozpoznávání řeči překladač Microsoft

Postupujte podle těchto kroků k registraci pro rozhraní API pro rozpoznávání řeči překladač společnosti Microsoft.

## <a name="1-sign-into-azure"></a>1: Přihlaste se k Azure   
- Nemáte účet? [Zaregistrujte si účet Microsoft Azure](http://azure.com/)
- Již máte účet? [Přihlásit se](http://portal.azure.com/)

## <a name="2-subscribe-to-microsoft-translator"></a>2: přihlášení k Microsoft překladač odběru     
- Po přihlášení do Azure, přejděte na [kognitivní služby](https://portal.azure.com/#create/Microsoft.CognitiveServices) části
- V části "Rozhraní API typu" Vyberte rozhraní API pro rozpoznávání řeči překladač
- V části cenová úroveň vyberte cenovou úroveň, která nejlépe vyhovuje vašim potřebám
    - Každé předplatné obsahuje volné vrstvy. Úroveň free má stejné vlastnosti a funkce jako placené plány a nemá datum vypršení platnosti.
    - Máte pouze jedno bezplatné předplatné ve vašem účtu
- Vyplňte zbytek formuláře a kliknutím na tlačítko Vytvořit
- Předplatné je připravený k okamžitému použití

## <a name="3-authentication-key"></a>3: ověřovací klíč    
- Načtení ověřovací klíč přechodem do nabídky všechny prostředky > klikněte na předplatné
- Hodnota klíče se používá pro ověřování, ponechte hodnotu klíče důvěrné. Hodnota klíče je nutné při vývoji aplikace
