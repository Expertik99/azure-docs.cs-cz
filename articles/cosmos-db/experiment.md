# <a name="azure-cosmos-db-documentation-experiment"></a>Experiment dokumentace Azure Cosmos DB

Azure Cosmos DB je globálně distribuované více modelu databáze. Protože můžete provést tak většinu s Azure Cosmos DB – vytváření graf k vytvoření databáze dokumentu nebo migraci dat z MongoDB, Cassandra nebo Azure Table storage – uspořádání dokumentace může být složité. Proto rádi bychom znali váš nápovědy.

Testujeme organizace Azure Cosmos Databázové tabulce obsahu a [cílová stránka](https://docs.microsoft.com/en-us/azure/cosmos-db/), jak je znázorněno v následujícím bitovým kopiím, a rádi bychom se získat váš názor na tyto navrhované změny.

## <a name="to-load-both-views"></a>Chcete-li načíst obou zobrazeních.

1. Chcete-li otevřít obě zobrazení vedle sebe, přejděte na [cílová stránka](https://docs.microsoft.com/en-us/azure/cosmos-db/) z oblíbených položek webového prohlížeče. 
2. Pomocí výše uvedených bitových kopií k určení, která verze se zobrazí, původní nebo experimentální. 
3. Otevřete okno prohlížeče inprivate nebo incognito a zatížení [cílová stránka](https://docs.microsoft.com/en-us/azure/cosmos-db/) v nové okno prohlížeče a znovu zjistit, která verze se zobrazí pomocí bitové kopie. 

   Pokud nemáte obou zobrazeních přítomen, zavřete a znovu otevřete nové okno se službou inprivate nebo incognito a zatížení [cílová stránka](https://docs.microsoft.com/en-us/azure/cosmos-db/) znovu. Může trvat několik pokusí získat obě verze načíst v samostatném systému windows. Aktualizace okna incognito nenačte jinou verzi, budete muset otevřete nové okno. 

4. Pak můžete nastavit velikost okna pro opravu vedle sebe nebo přepínat mezi zobrazeními k porovnání. 

5. Po vám, že možnost Seznamte se s organizace a rozložení experimentální prostředí, klikněte na tlačítko **experimentovat průzkum** k poskytnutí zpětné vazby níže uvedené tlačítko. Odeslání formuláře později než půlnoc 4 duben 2018 tak, aby zpětnou vazbu, můžete zkontrolovat a začlenit i metodu.
