# <a name="get-started-with-the-cognitive-capabilities-of-u-sql"></a>Začínáme s kognitivní možnosti U-SQL

## <a name="overview"></a>Přehled
Kognitivní možnosti U-SQL umožňují vývojářům používat inteligentní funkce put ve svých aplikacích velké objemy dat. 

K dispozici jsou následující kognitivních možností:
* Vytvoření bitové kopie: Rozpoznávání tváří [vzorku](https://github.com/Azure-Samples/usql-cognitive-imaging-ocr-hello-world)
* Vytvoření bitové kopie: Detekovat emoce [vzorku](https://github.com/Azure-Samples/usql-cognitive-imaging-emotion-detection-hello-world)
* Vytvoření bitové kopie: Zjišťovat objekty (označování) [vzorku](https://github.com/Azure-Samples/usql-cognitive-imaging-object-tagging-hello-world)
* Vytvoření bitové kopie: OCR (optické rozpoznávání znaků) [vzorku](https://github.com/Azure-Samples/usql-cognitive-imaging-ocr-hello-world)
* Text: Klíče frázi extrakce a analýza mínění [vzorku](https://github.com/Azure-Samples/usql-cognitive-text-hello-world)
