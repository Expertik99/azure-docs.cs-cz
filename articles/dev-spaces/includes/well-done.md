## <a name="well-done"></a>Skvělá práce!
Po dokončení Příručka Začínáme! Naučili jste se tyto postupy:

> [!div class="checklist"]
> * Vytvořte prostředí na základě Kubernetes v Azure, která je optimalizovaná pro vývoj.
> * Opakované vývoj kódu v kontejnerech pomocí VS Code a příkazového řádku.
> * Vývoj nezávisle dvě samostatné služby a použít Kubernetes zjišťování služby DNS k volání na jinou službu.
> * Produktivní vývoj a testování kódu v prostředí team.

Teď, když jste prozkoumali Azure Dev mezery, [sdílet vaše vývojová prostředí s seskupení](../how-to/share-dev-spaces.md) a věděli, jak snadno se dá spolupráce společně.
