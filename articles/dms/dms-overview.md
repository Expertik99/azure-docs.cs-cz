# <a name="what-is-the-azure-database-migration-service-preview"></a>Co je Azure databáze migrace služby Preview?
Služba Azure databáze migrace je plně spravovaná služba navržených k povolení bezproblémové migrace z více zdrojů databáze do platformy Azure dat s minimálními výpadky. Služba je ve verzi Public Preview, se zaměřuje na úsilí vývoj aktuálně:

- Spolehlivost a výkon.
- Iterační přidání dvojic cílového zdroje.
- Trvalá investice do bez třecí migrace.

## <a name="use-familiar-tools"></a>Používáte známé nástroje
Služba Azure migrace databáze integruje některé funkce našich existujících nástrojů a služeb.  Poskytuje zákazníkům komplexní a vysoce dostupné řešení. Služba Azure migrace databáze používá [Data migrace pomocníka](http://aka.ms/dma) zajistit assessment sestavy.  Tato doporučení vás provede změny, které jsou požadovány před provedením migrace. Je to na můžete provádět nápravy škod způsobených vyžaduje. Jakmile budete připraveni k zahájení procesu migrace, služba migrace databáze Azure provádí všechny přidružené kroky. Můžete aktivovat a zapomněli migrace projekty s pocit zjistit, které proces využívá osvědčených postupů, které určí společnost Microsoft.
