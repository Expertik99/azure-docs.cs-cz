# <a name="get-pre-approved-via-the-microsoft-azure-certified-program"></a>Získat předem schválené prostřednictvím programu Microsoft Azure Certified
Tento článek obsahuje pokyny k certifikaci Microsoft Azure, aby mohli publikovat řešení do Azure Marketplace a kvalifikaci pro marketing výhody. Podrobný přehled o programu Microsoft Azure Certified a jeho výhody, najdete v článku [ http://azure.com/certified ](http://azure.com/certified).

> [!IMPORTANT]
> Azure certifikační lze použít pouze v případě, že se chystáte vytvořit virtuální počítač nebo šablonu řešení pro Azure Marketplace.

## <a name="getting-started"></a>Začínáme
1. Prvním krokem v cestě k získávání řešení certifikaci a zařazený, nemá v Azure Marketplace je k vyplnění formuláře aplikace na [Microsoft Azure certifikované](https://createopportunity.azurewebsites.net) webovou stránku.
2. Pokud splňují kritéria způsobilosti a aplikace je schválená, spustíme práce s vámi na zaváděním řešení v Azure Marketplace.
3. Pokud pracujete s partnera účet manažera nebo partnera správce Directx, požádejte je určíme řešení pro certifikaci Azure programu, nebo přejděte ke [certifikaci Microsoft Azure](http://createopportunity.azurewebsites.net) webovou stránku, vyplňte formulář, aplikace a zadejte e-mailu partnera Account manažera nebo partnera Manager Directx v poli kontaktu sponzor Microsoft.

> [!IMPORTANT]
> **"Zákazník – od" zemích** – Chcete-li prodeje vašim službám na webu Microsoft Azure Marketplace, vaše registrovaná entity, musí být z jednoho z zemí schválené "zákazník – od". Toto omezení je výběr a zdanění z důvodů. Aktivně chceme v blízké budoucnosti rozbalte tento seznam zemí, tak si Nenechte ujít. Úplný seznam najdete v části 1b z [zásady zapojení Azure Marketplace](http://go.microsoft.com/fwlink/?LinkID=526833).
