# <a name="enable-lead-sharing"></a>Povolit sdílení realizace  
Zkontrolujte, jestli je zapnutá správa realizace vaší výpis na marketplace, tak, aby se vám zobrazilo kontaktní informace pro tyto cílové skupiny.

| Výkladní skříň | Cílová skupina |  
|:--- |:--- |  
| AppSource | obchodní zákazníků |  
| Azure Marketplace | Zákazníci |  

Zájemců poskytují základ dosah pro následující kampaně, prodejní pohyby pole pracovníků a informace o tom, jak je provádění vaší aplikace nebo nabízení. 

| Výkladní skříň | Kampaň |  
|:--- |:--- |  
| AppSource | Nurture |  
| Azure Marketplace | generování v vyžádání |  

Použijte následující postupy pro vaše zájemce.  
*   Zařazování zájemců a vyhodnocování zájemců jako prodejních příležitostí
*   Vydavatelských zájemců zadat prodejní kanál
*   Zarovnání činnosti spojené s vaše celková strategie na kampaň 

Zájemců ukazují zájmu zákazníka v následujícím seznamu. 

| Výkladní skříň | Výpis |  
|:--- |:--- |  
| AppSource | <table> <tr><td>Aplikace</td></tr> <tr><td>Konzultační služby</td></tr> </table> |  
| Azure Marketplace | <table> <tr><td>Nabídka</td></tr> <tr><td>Technologie</td></tr> </table> |  

Proces by měla mít zavedené zodpovědná za zájemců.  

| Výkladní skříň | Vede |  
|:--- |:--- |  
| AppSource | <table> <tr><td>vaše CRM</td></tr> <tr><td>Pole Prodejce Microsoft CRM</td></tr> </table> |  
| Azure Marketplace | <table> <tr><td>Azure Marketplace</td></tr> <tr><td>Pole Prodejce Microsoft CRM</td></tr> </table> |  

Zájemců v obchodní poutače na webu marketplace se liší od prodejce kvalifikovaný zájemců Microsoft, které jsou funkce programu společné zákazník.  
*   Další informace o tom, jak přístup programu společné prodává navštívit podporovat vaše obchodní s stránku Microsoft se nachází na [povýšit--business s Microsoftu](./promote-your-business-with-microsoft.md).
