# <a name="what-is-appsource"></a>Co je AppSource
AppSource umožňuje partnerům Microsoftu zjistitelnost své technologie a služby zákazníkům prostřednictvím podporovaných Microsoft online výkladní skříň. Jako vydavatel AppSource můžete zaujmout uživatele obchodní na AppSource najít, zkuste a get-obchodní aplikace SaaS, jakož i služby implementace pomoct jednotky obchodní výsledky a snižte hodnotu doby: 

| Potřeba zákazníka       | AppSource          | 
|:---------------------------------------- |:----------------------------------------------------- |
| **Hledá podniková řešení, které pracují s produkty společnosti Microsoft již používají**    | Umožňuje partnerům třetí strany aplikací a služeb můžete použít k rozšíření cloudových aplikací společnosti Microsoft a technologie. |
| **Schopnost snadno najít právo vpravo řešení nebo implementace služby.**     | Poskytuje centralizované ke zjištění, zkušební verze a získat aplikací a služeb, doplňky a mnoho dalších. |
| **Průmyslové obchodní řešení pro jejich konkrétní obchodní problémy podpory**   | Poskytuje oborový dokončení začátku do konce řešení, která pomůžou konkrétní požadavky na adresu napříč mnoha odvětví. |
| **Aplikacemi a pomáhá zvýšit produktivitu, efektivitu a podnikových statistik**       | Poskytuje aplikace pro obchodních, včetně služeb zákazníkům, oddělení lidských zdrojů, operace a mnoho dalších. |
| **Přizpůsobí aplikace jejich jedinečných situaci partnera zkušeného implementace**     | Zadejte, katalog pro řešení založená na Dynamics 365, Power BI, PowerApps a 3. stran aplikace, které jsou k dispozici na AppSource pomohou podnikoví uživatelé najít konzultační služby navržený tak, aby předvídatelný výstupy konzultace ohledně nabídky služeb. |

## <a name="appsource-publishing"></a>Publikování AppSource
Prostřednictvím AppSource můžete seznam aplikace nebo konzultační nabídka, která pomáhá pokrývat zákazníka při používajících produkty společnosti Microsoft, například Office 365, Dynamics 365, Power BI a výkonu aplikací. Následující nabídky jsou k dispozici na webu Marketplace AppSource: 
*   **Dynamics 365 Finance a operace**: Zdroj organizace plánování řešení (ERP). Odlišné řešení jsou Enterprise Edition pro střední pro velké společnosti a Business Edition pro malé a střední firmy.
*   **Dynamics 365 pro zákaznické Engagement**: řešení správy (CRM) vztah zákazníků, které zahrnuje prodej, oddělení služeb zákazníkům, pole služby a aplikace projektu služby Automation.
*   **Služba spravovaných Dynamics NAV**: Microsoft **Dynamics NAV** je prostředek enterprise plánování suite softwaru (ERP) pro středně velkých organizací. Nabízí služba specializované funkce pro výrobní, distribuci, government, maloobchodní a dalších odvětví.
*   **Power BI**: řešení vizualizace obchodní analýzy.
*   **Konzultace ohledně nabízí**: služby poskytované společností Microsoft specializuje partnery pro zákazníky, pochopit, si můžete vyzkoušet a implementovat řešení technologie.
*   **Cortana Intelligence**: The Cortana Intelligence Suite je kolekce nezávislé, ale plně integrované dat a analytické platformě nástroje, které nabízí Azure.
*   **Office 365**: moderní síti na pracovišti v cloudu. Zdarma Spolupracujte s Microsoft Word, PowerPoint, aplikace Excel a OneNote.

## <a name="azure-active-directory-integration"></a>Integrace Azure Active Directory
Některé aplikace storefront AppSource nutných k integraci se službou Azure Active Directory (Azure AD) k publikování. Integrace aplikací s Azure AD je dobře zdokumentovat; Společnost Microsoft poskytuje několik sady SDK a další prostředky podle požadavků. 

| Název nabídky | Vyžaduje integrace Azure AD | Poznámky |
|:--- |:---:|:--- |
| Dynamics 365 for Operations | Ne | Nabídku na základě licencí nevyžaduje integrace se službou Azure AD |
| Dynamics 365 pro finanční | Ne | Nabídku na základě licencí nevyžaduje integrace se službou Azure AD |
| Dynamics 365 pro zákaznické Engagement | Ne | Nabídku na základě licencí nevyžaduje integrace se službou Azure AD |
| Dynamics NAV spravované služby | Ne | Nabídku na základě licencí nevyžaduje integrace se službou Azure AD |
| Power BI | Ne | Nabídku na základě licencí nevyžaduje integrace se službou Azure AD |
| Konzultace ohledně nabídky | Ne | Partner poskytující prostředky, ne přes webové prostředí jsou koordinované služby |
| Cortana Intelligence | Ano | Azure AD je osvědčeným postupem zajistit bezproblémové zákazníka, zkušeností a jednotky, které vede nejvyšší kvality |
| Aplikace SaaS (dříve webové aplikace) | Upřednostňovaný | Azure AD je osvědčeným postupem zajistit bezproblémové zákazníka, zkušeností a jednotky, které vede nejvyšší kvality |
