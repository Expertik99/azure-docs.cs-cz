# <a name="what-is-appsource"></a>Co je AppSource
AppSource umožňuje partnerům Microsoftu zjistitelnost jejich technologie a služby zákazníkům prostřednictvím online prezentace podporovaný společností Microsoft. Jako na vydavatele AppSource můžete zapojení uživatelů business na AppSource a vyhledat, zkoušet a get-obchodní aplikace SaaS, jakož i implementaci služby a pomůže zlepšit výsledky firmy a snižte hodnotu času: 

| Potřeba zákazníka       | AppSource          | 
|:---------------------------------------- |:----------------------------------------------------- |
| **Hledáte obchodních řešení, které fungují s produkty Microsoftu již používají**    | Umožňuje partnerům používat aplikace třetích stran a služby pro rozšíření společnosti Microsoft cloudových aplikací a technologie. |
| **Schopnost snadno najít vpravo požadovanou službu řešení nebo implementace.**     | Poskytuje univerzálním hledat, zkoušet, a získat aplikací a služeb, doplňky a mnoho dalších. |
| **Specifické pro odvětví podnikových řešení pro jejich konkrétní obchodní problémy**   | Poskytuje dokončení začátku do konce oborová řešení ke konkrétní požadavky na adresu v mnoha oborech. |
| **Aplikace, které jí pomohou zlepšit produktivitu, efektivitu a obchodní přehledy**       | Poskytuje aplikace pro obchodní, včetně služby zákazníkům, HR, operací a mnoho dalších. |
| **Zkušení implementace partnera, který přizpůsobí aplikace jejich jedinečných situaci**     | Zadejte, katalog consulting services nabídky pro řešení založená na Dynamics 365, Power BI, PowerApps a aplikace 3rd třetích stran, které jsou k dispozici na AppSource pro obchodní uživatelé snáze našli konzultační služby navržené pro zajištění předvídatelné výsledky. |

## <a name="appsource-publishing"></a>Publikování na AppSource
Prostřednictvím AppSource můžete vytvořit seznam aplikací nebo konzultační nabídka, která pomáhá splnit požadavky zákazníků při používání produktů společnosti Microsoft, jako je Office 365, Dynamics 365, Power BI a Power Apps. Tyto nabídky jsou dostupné na webu Marketplace AppSource: 
*   **Dynamics 365 pro Finance and Operations**: podnikových zdrojů (ERP) řešení plánování. Enterprise Edition pro střední pro velké společnosti a Business Edition pro malé a střední podniky jsou rozlišené řešení.
*   **Dynamics 365 for Customer Engagement**: řešení pro správu (CRM) vztah se zákazníky zahrnuje prodej, služby zákazníkům, služby a aplikace automatizace projektových služeb.
*   **Spravovaná služba pro Dynamics NAV**: Microsoft **Dynamics NAV** je podnikových zdrojů (ERP) softwarový balík pro středně velké organizace plánování. Tato služba nabízí specializované funkce pro výrobu, distribuce, státní správy, maloobchodní a další obory.
*   **Power BI**: řešení obchodní analýzy vizualizace.
*   **Consulting nabízí**: služby poskytované společností Microsoft partnerům, aby zákazníkům pomohla pochopit, vyzkoušejte si a implementovat specializovaný technologickými řešeními.
*   **Cortana Intelligence**: The Cortana Intelligence Suite je kolekce nezávislé, ale nabízí plně integrovaná data a analytické platformě nástroje Azure.
*   **Office 365**: moderní pracoviště v cloudu. Spolupracujte bezplatné aplikace Microsoft Word, PowerPoint, Excel a OneNote.

## <a name="azure-active-directory-integration"></a>Integrace Azure Active Directory
Některé aplikace prezentace AppSource se musí integrovat s Azure Active Directory (Azure AD) k publikování. Integrace aplikací s Azure AD je dobře zdokumentovaná; Společnost Microsoft poskytuje několik sad SDK a další prostředky podle svých požadavků. 

| Název nabídky | Vyžaduje integraci Azure AD | Poznámky |
|:--- |:---:|:--- |
| Dynamics 365 for Operations | Ne | Nabídka založená na licenci nevyžaduje, aby integrace služby Azure AD |
| Dynamics 365 pro Finance | Ne | Nabídka založená na licenci nevyžaduje, aby integrace služby Azure AD |
| Dynamics 365 for Customer Engagement | Ne | Nabídka založená na licenci nevyžaduje, aby integrace služby Azure AD |
| Dynamics NAV spravované služby | Ne | Nabídka založená na licenci nevyžaduje, aby integrace služby Azure AD |
| Power BI | Ne | Nabídka založená na licenci nevyžaduje, aby integrace služby Azure AD |
| Konzultaci s nabídkami | Ne | Služby jsou koordinovaný partner, ne prostřednictvím webového prostředí |
| Cortana Intelligence | Ano | Azure AD je osvědčeným postupem zajistit bezproblémové zákaznické prostředí a jednotky, v nejvyšší kvalitě potenciálních zákazníků |
| Aplikace SaaS (dříve webové aplikace) | Upřednostňovaný | Azure AD je osvědčeným postupem zajistit bezproblémové zákaznické prostředí a jednotky, v nejvyšší kvalitě potenciálních zákazníků |
