# <a name="supported-azure-database-for-mysql-server-versions"></a>Podporované Azure databáze MySQL verzí serveru
Azure databáze pro databázi MySQL byla vyvinuta z [MySQL Community Edition](https://www.mysql.com/products/community/), pomocí modulu InnoDB.  Databáze pro databázi MySQL Azure aktuálně podporuje následující verze:

## <a name="mysql-version-5635"></a>MySQL Version 5.6.35
Podívejte se na MySQL [dokumentace] (https://dev.mysql.com/doc/relnotes/mysql/5.6/en/news-5-6-35.html) Další informace o vylepšení a opravy v MySQL 5.6.35.

## <a name="mysql-version-5718"></a>Verze databáze MySQL 5.7.18
Podívejte se na MySQL [dokumentace] (https://dev.mysql.com/doc/relnotes/mysql/5.7/en/news-5-7-18.html) Další informace o vylepšení a opravy v MySQL 5.7.18.

## <a name="managing-updates-and-upgrades"></a>Správa aktualizací a upgradů
Azure databáze pro databázi MySQL automaticky spravuje opravy pro vedlejší verze aktualizace. Upgrady hlavní verzi, například upgradu z databáze MySQL 5.6 na MySQL 5.7, nejsou podporované.
