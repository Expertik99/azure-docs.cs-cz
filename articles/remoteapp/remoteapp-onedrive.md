# OneDrive pro firmy a Azure RemoteApp

> [AZURE.IMPORTANT]
> Azure RemoteApp se přestává používat. Podrobnosti najdete v tomto [oznámení](https://go.microsoft.com/fwlink/?linkid=821148).

Synchronizační agent OneDrivu pro firmy se nepodporuje s Azure RemoteAppem.

OneDrive pro firmy můžete zahrnout do kolekce a používat s Office 365.
