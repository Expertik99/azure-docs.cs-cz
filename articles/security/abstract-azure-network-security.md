# <a name="azure-network-security"></a>Zabezpečení sítě Azure
## <a name="abstract"></a>Abstraktní
Síť Azure services maximalizovat flexibilitu, dostupnost, odolnost proti chybám, zabezpečení a integrity návrhu. Tento dokument white paper poskytuje podrobné informace o síťové funkce Azure. Také popisuje, jak zákazníci mohou používat funkce nativní zabezpečení v Azure k ochraně svých prostředků informací.
Zamýšlenou cílovou skupinou pro tento dokument white paper patří:
- Technické správci, správci sítě a vývojáři, kteří hledají řešení zabezpečení, které jsou dostupné a podporované v Azure.
- Malé a střední podniky nebo vedoucí proces pracovníky, kteří chtějí podrobný přehled technologie Azure a služby, které se týkají zabezpečení sítě ve veřejném cloudu Azure.
