# <a name="microsoft-security-response-center"></a>Středisko Microsoft Security Response Center

Microsoft Security Response Center (MSRC) je vedla pomocí některé z nejvíc zkušeného zabezpečení odborníky na světě. Jejich identifikovat, monitorování, reagovat a vyřešení incidentů zabezpečení a místní a cloudové ohrožení zabezpečení hodin každý den v roce.

Kromě nepřetržitá práce má týmem střediska MSRC počet prostředků, které jsou k dispozici, takže můžete pochopit, jak zabezpečit prostředky Azure a nasazení efektivnější.

## <a name="the-msrc-blog"></a>Blog střediska MSRC

[Střediska MSRC blog](https://blogs.technet.microsoft.com/msrc/) je místo, kde získat nejnovější informace na střediska MSRC to vás pomáhá chránit proti hrozbám cloudu.

## <a name="white-papers"></a>Dokumenty white paper

Střediska MSRC publikovali řadu [dokumenty white paper](https://technet.microsoft.com/library/bb969102.aspx) , snadněji pochopit, co dělají a jak se. Některé poskytují přehled o tom, jak jsme zabezpečení cloudu Microsoftu a obsahovat užitečné informace o tom, jak můžete použít stejné konfigurace zabezpečení.

## <a name="security-researcher-engagement-and-bounty-programs"></a>Zapojení zkušeností zabezpečení a hojnosti programů

Střediska MSRC podporuje spolupráci a vztahů s výzkumných pracovníků zabezpečení globálně k posunutí zabezpečení produktu společnosti Microsoft.

Programy společnosti Microsoft hojnosti platit výzkumných pracovníků pro:

- Nové zneužití techniky
- Obranným nápady, která snižuje riziko zneužití nové
- Identifikace kritické chyby zabezpečení v Microsoft místní a cloudové softwaru.
