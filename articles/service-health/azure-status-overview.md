# <a name="azure-status"></a>Stav Azure
[Azure stav](https://azure.microsoft.com/status/) poskytuje globální zobrazení stavu služby Azure. S Azure stav můžete získat nejnovějších informací o dostupnosti služby. Všem uživatelům přístup k Azure stav a můžete zobrazit všechny služby, které sestavy jejich stav služby. 

![Stránku stavu Azure](./media/azure-status-overview/azure-status.PNG)

## <a name="azure-status-updates"></a>Aktualizace stavu Azure
Jako stavu služby Azure změny získá aktualizaci stránky Azure stav v reálném čase. Pokud necháte Azure stavové stránce otevřete, můžete řídit rychlost, jakou stránce aktualizuje nová data. V horní části se zobrazí čas poslední aktualizace stránky.

![Aktualizace stavu Azure](./media/azure-status-overview/update.PNG)

## <a name="azure-status-history"></a>Historie stavu Azure
Zatímco Azure stavové stránce se zobrazí vždy nejnovější informace o stavu, můžete zobrazit pomocí starší události [Azure stavové stránce Historie](https://azure.microsoft.com/status/history/).

Najdete v něm události až do 90 dnů v minulosti, včetně předběžný hlavní příčinu, omezení rizik a další kroky pro Azure.

## <a name="rss-feed"></a>Informační kanál RSS
Také poskytuje Azure stav [informačního kanálu RSS](https://azure.microsoft.com/status/feed/) změny stavu služby Azure, které se můžete přihlásit k odběru.
