# <a name="manage-virtual-machine-extensions"></a>Správa rozšíření virtuálního počítače
Popisuje, jak najít, přidávat, upravovat nebo odebírat rozšíření virtuálního počítače se systémem Windows, nebo virtuální počítače s Linuxem v Azure.

> [!IMPORTANT] 
> Azure má dva různé modely nasazení pro vytváření a práci s prostředky: [Resource Manager a klasický](../../../resource-manager-deployment-model.md). Tento článek se zabývá pomocí modelu nasazení Classic. Microsoft doporučuje, aby byl ve většině nových nasazení použit model Resource Manager. Informace o rozšíření virtuálního počítače pomocí modelu Resource Manager najdete v tématu [zde](../extensions-features.md?toc=%2fazure%2fvirtual-machines%2flinux%2ftoc.json).
