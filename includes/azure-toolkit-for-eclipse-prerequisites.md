## <a name="prerequisites"></a>Požadavky
Sada nástrojů Azure pro Eclipse vyžaduje následující softwarové komponenty:

* Integrované vývojové prostředí Eclipse pro vývojáře v jazyce Java EE, Mars, nebo novější. Si můžete stáhnout z [Eclipse webu](http://www.eclipse.org/downloads/).

* Java Developer Kit (JDK), verze 1,8 nebo novější.

* Operační systém. Sada nástrojů Azure pro Eclipse byl otestován v následujících operačních systémů:
  
  * Windows 10, Windows 8.1, Windows 8 a Windows 7
  * Windows Server 2012 R2, Windows Server 2012, Windows Server 2008 R2 a Windows Server 2008
  * [Mac OS X](http://www.apple.com/osx) verze "Yosemite" a novější
  * [Ubuntu Linux](http://www.ubuntu.com) verzi 14, 15 a 16

Další informace najdete v tématu [nástrojů Azure pro Eclipse](http://marketplace.eclipse.org/content/azure-toolkit-eclipse) stránku v prostředí Eclipse Marketplace.
