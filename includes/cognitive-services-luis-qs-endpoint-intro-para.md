V tomto rychlém startu použijte k určení záměru uživatele z textu lze aplikaci dostupnou LUIS veřejné. Odeslání záměru uživatele jako text do veřejné aplikace HTTP předpovědi koncového bodu. V koncovém bodě LUIS platí veřejné aplikační model pro analýzu textu přirozeného jazyka, což znamená, určení celkového záměr a extrahování dat, které jsou relevantní pro domény předmětu vaší aplikace. 

Tento rychlý start využívá koncový bod rozhraní REST API. Další informace najdete v tématu [koncových bodů dokumentace k rozhraní API](https://westus.dev.cognitive.microsoft.com/docs/services/5819c76f40a6350ce09de1ac/operations/5819c77140a63516d81aee78).

Pro účely tohoto článku budete potřebovat bezplatný [LUIS](http://www.luis.ai) účtu.
