V případě potřeby nainstalujte modul Azure PowerShell pomocí pokynů v [prostředí Azure PowerShell průvodce](https://docs.microsoft.com/powershell/azureps-cmdlets-docs/)a poté spusťte `Login-AzureRmAccount` vytvořit připojení s Azure. Také je potřeba mít veřejný klíč SSH s názvem `id_rsa.pub` v adresáři .ssh uživatelského profilu.
