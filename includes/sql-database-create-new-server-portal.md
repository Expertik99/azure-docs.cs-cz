## <a name="create-an-azure-sql-database-logical-server"></a>Vytvoření logického serveru Azure SQL Database
Pomocí následujícího postupu můžete na portálu Azure vytvořit logický server Azure SQL Database.

1. Pokud nejste připojeni, připojte se k webu [Azure Portal](http://portal.azure.com).
2. Klikněte na **Nové**, klikněte na **Data + úložiště**, a vyhledejte **SQL Database**.
3. Kliknutím na **SQL Database** otevřete okno SQL Database.
4. Zadejte hodnoty následujících vlastností serveru:
   
   * Název serveru
   * Přihlašovací jméno správce serveru
   * Heslo
   * Předplatné (jenom pokud máte více předplatných)
   * Skupina prostředků (nová nebo stávající)
   * Umístění
5. Klikněte na možnost **Vytvořit**. V oznamovací oblasti uvidíte, že nasazení bylo zahájeno.
6. Než budete pokračovat na další krok, počkejte, až se nasazení dokončí.
