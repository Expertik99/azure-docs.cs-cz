## <a name="supported-ssh-key-formats"></a>Podporované formáty klíče SSH

Azure aktuálně podporuje SSH protokolu 2 RSA (SSH-2) dvojice veřejného a privátního klíče, které s minimální délkou 2 048 bitů. Ostatní formáty klíče například ED25519 a ECDSA nejsou podporovány.
